4. Tournament System (Core Feature)
4.1 Tournament Creation (Admin)

Admin defines:

Game (PUBG, Valorant, MLBB, etc.)

Tournament type (Squad, Duo, Solo)

Maximum teams (e.g., 32)

Team size (e.g., 5 members)

Registration deadline

Rules & point system

4.2 Team-Based Registration Flow (Refined)
Step 1: User Login

Only logged-in users can:

Create teams

Join teams

Register tournaments

Step 2: Team Creation

Any user can create a team

Creator becomes Team Captain

Team is game-specific (PUBG team ≠ Valorant team)

Team status:

Incomplete (less than required members)

Complete (ready for tournament)

Step 3: Team Formation

Team captain can:

Search users

Send invite link

Accept/reject join requests

Remove members

Team requirements example:

PUBG Squad → 5 players

1 Captain

4 Members

Step 4: Tournament Registration

Only Team Captain can register

System checks:

Team is complete

Team is not already registered

Once registered:

Team slot is locked

4.3 Automatic Registration Control

When team limit (e.g., 32) is reached:

Registration closes automatically

Tournament status becomes “Registration Closed”

4.4 Fixture & Bracket Generation

System auto-generates:

Group stage / Knockout brackets

Displayed on:

Tournament detail page

Game-specific page (PUBG, Valorant)
